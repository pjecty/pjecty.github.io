카카오페이 API를 이용해 결제기능 만들기


detail 페이지 내 HTML 버튼(button)에 대한 이벤트 처리(onclick)를 정의해 함수가 호출되고 해당 함수에 ${DTO.id} 값이 인자로 전달되며 결제정보 팝업이 생성되도록 처리했습니다.
주소찾기 버튼을 눌렀을 때 반응은 다음(kakao)우편번호 서비스 api를 사용했습니다.

충전하고 싶은 금액을 input 박스에 입력 후 포인트충전 버튼을 눌렀을 때 KakaoPay QR 팝업이 열리고 결제가 가능하도록 설정했습니다.
충전받은 금액을 변수로 저장하여 정수로 변환한 값이 할당되도록 설정했습니다. 그리고 ajax를 활용하여 포인트가 충전되면 (사용 가능한 포인트)에 비동기로 출력되도록 설정했습니다.
