JPA를 활용해 CRUD 게시판 기능 구현 (Create)


CRUD는 컴퓨터 프로그래밍에서 자주 사용되는 용어로, 데이터의 생명 주기를 다루는 (Create, Read, Update, Delete) 의미합니다.

중고 마켓 게시글 등록에 필요한 기본적인 정보를 입력받는 form 입니다.
여러 개의 파일을 선택할 수 있도록 multiple 속성을 정의 했습니다. placeholder 속성으로 사용자가 입력 필드에 무엇을 적어야하는지 힌트를 제공합니다.
전체적으로 input값의 유효성을 검사하고 결과를 화면에 표시하는 정규식을 사용했습니다. 올바르게 작성되었다면 green, 올바르지 않다면 red 컬러의 텍스트를 출력하게 했습니다.

form 에서 전달받은 값은 컨트롤러에서 POST 요청을 처리하며 DTO값을 매개변수로 받아 Service에서 파일 유무를 판단하고
Spring Data JPA에서 사용되는 리포지토리 인터페이스로 엔티티와 관련된 데이터베이스 작업을 자동으로 수행하게해 생성이 가능합니다.

JPA를 활용해 CRUD 게시판 기능 구현 (Read)


중고 마켓에 등록된 게시글을 모든 상품을 조회하는 리스트입니다.
Repository.findAll를 사용하여 데이터베이스에서 모든 상품 정보를 조회합니다. 이때, 상품은 생성된 시간을 기준으로 내림차순으로 정렬되도록 설정했습니다.
Detail 페이지를 생성해 list의 상품을 누르면 id라는 매개변수를 받아 새로운 URL을 만듭니다. 사용자가 클릭한 상품의 상세 정보 페이지로 이동할 수 있도록 했습니다.
