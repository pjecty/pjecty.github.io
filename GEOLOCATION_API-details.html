Geolocation API는 자바스크립트 프로그램이 사용자의 실제 위치를 브라우저에게 요청할 수 있도록 해준다.


Geolocation API는 navigator.geolocation 객체를 통해 사용할 수 있습니다

if (navigator.geolocation) { 이 부분은 브라우저가 Geolocation API를 지원하는지 확인합니다.
navigator.geolocation은 브라우저의 Geolocation API에 액세스하는 객체입니다. 이 조건문은 브라우저가 Geolocation을 지원하는 경우에만 실행됩니다.

navigator.geolocation.watchPosition 함수는 위치 정보를 주기적으로 업데이트하는데 사용됩니다.
새로운 위치 정보가 이 함수의 콜백 함수로 전달됩니다. 이 콜백 함수는 pos 매개변수를 받고, 해당 위치의 위도와 경도를 #latitude와 #longitude 요소에 표시

btnStop"라는 버튼이 클릭될 때 위치 정보를 더 이상 업데이트하지 않도록 설정한 id를 전달하고 지원하지 않는 브라우저를 사용할 시 알람을 띄워줍니다.
